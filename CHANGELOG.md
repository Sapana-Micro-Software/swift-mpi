# Changelog

All notable changes to SwiftMPI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial framework release
- Complete MPI implementation in pure Swift
- Point-to-point communication (blocking and non-blocking)
- Collective operations (broadcast, reduce, gather, scatter, etc.)
- Communicator management
- Datatype and operation definitions
- Comprehensive test suites (XCTest and SwiftTesting)
- Full documentation

### Framework Features
- Dynamic library distribution
- Swift Package Manager support
- Multi-platform support (macOS, iOS, tvOS, watchOS)
- Type-safe Swift API
- Comprehensive error handling
